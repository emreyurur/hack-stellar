THE PROBLEM

Stellar's DeFi ecosystem has $300M+ in TVL — but it's fragmented, opaque, and inaccessible.

  • Users need 3+ separate apps to access Blend, Soroswap, and Aquarius — no single view of positions.
  • 70%+ of wallets hold idle dust trustlines earning 0%. No tool to batch-sweep them into yield.
  • There is no trust signal for pools — users can't evaluate liquidity, audit status, or protocol age before committing funds.
  • Managing LP positions is complex: impermanent loss, reserve ratios, multi-step flows — nothing is explained.

THE SOLUTION

Terminal8 is a unified DeFi routing and position management interface built natively on Stellar Soroban.

  • Batch Swap Engine — sweep all dust trustlines into USDC in one PathPaymentStrictSend transaction.
  • Pool Reputation Score (0–100) — every pool rated across Liquidity, Age, Audit Status, and Activity. Like Ethos, but for Stellar DeFi.
  • Risk Profiling Quiz — 3-question onboarding personalizes pool recommendations: Conservative, Moderate, or Aggressive.
  • Yield Bundles — one-click multi-protocol allocation. Conservative Bundle splits into Blend lending in a single flow.
  • Developer CLI Terminal — withdraw 1 --full, pools, balance — real Freighter transactions from the command line.
Think

The Bloomberg Terminal for Stellar DeFi — powerful for developers, clear for everyone else.

TECHNICAL
WORKFLOW

01
Connect & Profile
Freighter wallet connects via @stellar/freighter-api. A 3-question risk quiz profiles the user (Conservative / Moderate / Aggressive) and personalizes all recommendations.
02
Review Trust Score
Pools are scored 0–100 across Liquidity (40), Age (20), Audit (20), and Activity (20). Expandable breakdown shows each axis — deterministic, verifiable on-chain in v0.2.
03
Supply or Bundle
Select a pool or execute a Yield Bundle. Blend supply calls PoolContractV2.submit() on Soroban. Soroswap LP calls the Router.add_liquidity() contract directly.
04
Manage via CLI
Positions tracked live with estimated yield. Power users manage via terminal: $ positions, $ withdraw 1 --full, $ pools — each triggers a real Freighter transaction.
